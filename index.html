Computer Vision "Go" Scoring System

Go is an abrstract strategy game similar to chess in that there are pieces on a board. After completing a game, it is difficult to score so I set out to make a tool that could score a bord automatically. I used a webcam to take a picture of the board and manipulated it into a square with linear algebra. After, I identified the corners and interpolated where the pieces were. Then I used a heuristic to determine the board state and then applied a graph theorey algorithm to score the board. After scoring, I used a projector to project an overlay back on to the board. I did this as a personal project, not for any class, and all in Python.

Codenames AI

Codenames is a word association game. There are 25 words on a table and each team captain has a set of words that they need to link together with one word hints for their team to guess. For example in the picture above, the red team needs to link together the words "pirate" and "face" so they could possibly say the word "eyepatch" since it is related to both. I wanted to make a program that could find these hints automatically. I used the library Word2Vec which represents every word as a 50+ dimensional vector. It has cool properties like if you take the vector for king, subtract the vector for man, and add the vector for woman, you have a vector that is extremely similar to the vector for queen. By taking the cosine distance between any two vectors you can get a rough approximation of how similar they were so I wrote a program that would calculate word similarity across groups of words to find the optimal hint.
